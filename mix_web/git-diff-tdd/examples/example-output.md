# 技术设计文档

## 一、背景

本次变更用于支持 Profile 创建接口新增 proxy_id 字段，
以支持代理自动绑定功能。

风险等级：中

---

## 二、接口变更说明

| 模块 | 控制器 | 方法 | 变更类型 | 是否破坏兼容 |
|------|--------|------|----------|--------------|
| fbcc | ProfileController | actionCreate | 新增参数 | 否 |

---

## 三、参数对比

### 旧版本

{
    "name": "string"
}

### 新版本

{
    "name": "string",
    "proxy_id": "int"
}

兼容性说明：

- proxy_id 非必填
- 老客户端不受影响

---

## 六、错误码审查

新增错误码：

ErrorCode::PROXY_NOT_FOUND

- 已定义于 ErrorCode.php
- 错误信息为英文
- 无硬编码错误码

---

## 七、风险分析

1. 数据一致性风险：低
2. 性能风险：无
3. 回滚复杂度：低

---

## 八、部署方案

1. 无需 DB 迁移
2. 直接灰度发布
3. 若异常，直接回滚代码
